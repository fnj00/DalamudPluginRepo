Custom FFXIVQuickLauncher plugin repository.

Installation
image

First open the Dalamud Settings menu in the game, accessible through typing /xlsettings in the chat or through the button at the bottom of the Plugin Installer.

Then follow the image:

Click the "Experimental" tab.
Paste https://raw.githubusercontent.com/fnj00/DalamudPluginRepo/master/pluginmaster.json into the empty box at the bottom of the "Custom Plugin Repositories" section (the last box will always have a + next to it).
Click the + button.
Click the 💾 button.
Now these plugins will appear inside of the Plugin Installer, accessible through typing /xlplugins in the chat.
